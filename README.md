exercise 1 summary:
### Create angular2 app:
```$ ng new my-app```
```$ cd my-app```
```$ ng serve```

goto [http://localhost:4200] and see if app works.

### Deployment
```$ ng build --prod```
```$ npm install -g firebase-tools```
```$ firebase login```
```$ firebase init```
```$ firebase deploy```
